# ADMETChallenge_PXR
Activity Track
A multi-agent ADMET activity prediction pipeline that featurizes molecules with full RDKit descriptors (~219), Morgan fingerprints, and 3-component PCA chemical-space coordinates, trains an XGBoost model optimized via Optuna Bayesian hyperparameter search, and leverages Claude Sonnet 4.6 for intelligent agent orchestration and result interpretation.
